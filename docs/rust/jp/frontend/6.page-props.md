# PageProps

画面の表示で使用されるデータを、PageProps 構造体に保持します。

https://github.com/mnitta220/microservices-demo-rust/blob/03e514d216f8a00fc9b7c4710f1f4bb55c05592d/src/frontend/src/pages/page.rs#L11-L41

Option 属性になっている項目は、その画面で使われる場合のみ、Some(T)がセットされ、使われない場合は None になります。  
Component の write メソッドには、PageProps への参照が渡されます。
